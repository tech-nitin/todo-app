# Todo App

A simple Todo application built using Vanilla JavaScript.

## Features
- Add, edit, delete tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)
- Clear completed tasks
- LocalStorage support

## Tech Stack
- HTML
- CSS
- JavaScript
